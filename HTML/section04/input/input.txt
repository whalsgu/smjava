[input tag]
입력 항목

(기본형)
<input type="유형" [속성="속성값"]>

(속성:id)
1. 폼의 항목(요소)를 구분하는 역할
2. label과 연결해서 제목을 붙임
3. css에서 id를 이용해 스타일일 적용
4. id 속성값은 문자로 구성되며 공백없어야 한다.

(type)
1. text : 한 줄까지 텍스트 입력 상자
2. hidden : 사용자에게 보이지 않지만 서버로 보내지는 값
3. search : 검색 상자
4. tel : 전화번호 입력 필드
5. url : url 주소 입력 필드
6. email : 메일 주소 입력 필드
7. password : 비밀번호 입력 필드
8. datetime : 국제표준시(UTC)로 설정된 날짜와 시간 입력 필드
9. datetime-local : 사용자 지역의 날짜와 시간 입력 필드
10. date : 사용자 지역 기준 날짜(연,월,일) 입력 필드
11. month : 사용자 지역 기준 날짜(연,월) 입력 필드
12. week : 사용자 지역 기준 날짜(연,주) 입력 필드
13. time : 사용자 지역 날짜(시,분,초, 분할 초) 입력 필드
14. number : 숫자를 조절하는 화살표
15. range : 숫자를 조전하는 슬라이드 막대
16. file : 파일 첨부 버튼
17. submit : 서버 전송 버튼
18. image : submit 버튼 대신 사용할 이미지
19. reset : 리셋 버튼
20. button : 버튼
21. color : 색상 표
22. checkbox : 체크 박스 버튼
23. radio : 항목 선택 버튼

Table Caption 태그

테이블의 캡션 태그는 표에 제목을 붙이는데 사용한다.
제목은 표의 위쪽 중앙에 표시가 된다.

다음 표는 프로그램 언어별 용도에 대한 것입니다. 내용은 자바언어는 오라클 사이트에서 제공하며 용도는 응용프로그램 개발을 하는데 최적화된 도구를 제공합니다. C/C++은 gnu.org 사이트에서 제공하며 시스템프로그램을 위한 언어로서 성능과 디바이스와 같은 영역의 개발에 사용됩니다.

Preformat(pre) 태그

Preformat(pre) 태그는 공백을 한 개만 표시한다.
문자열 사이에 여러 공백이 다음과 같이 있더라도 하나만 유효하다.
blockquote 처럼 들여쓰기가 된다.
프로그램 소스와 같은 코드들을 표현할 때 유용하다. 

        
[ Hello.java ]

        class Hello {
            public static void main(String[] args) {
                // 공백이 있는 그대로 출력된다.
                System.out.println("Hello, Java    !!!"); 
            }
        }

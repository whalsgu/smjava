순서 있는 목록 - ordered list

목록의 숫자 표기 방법이나 시작하는 숫자(기본값) 등을 표시

ol 태그에 속성값을 부여하여 다양한 형태의 숫자의 종류를 표시할 수 있다.

  • 속성값(1) : 숫자(기본값)
  • 속성값(a) : 영문 소문자
  • 속성값(A) : 영문 대문자
  • 속성값(i) : 로마숫자 소문자
  • 속성값(I) : 로마숫자 대문자
  • start 속성은 시작값을 조정
  • reversed 속성은 항목을 역순으로 표시

시작값이 7부터 시작

  1. 항목1
  2. 항목2
  3. 항목3

역순으로 표시

  1. 항목1
  2. 항목2
  3. 항목3

시작값이 10부터 역순으로 표시

  1. 항목1
  2. 항목2
  3. 항목3

[스타일 우선순위]
1. 사용자 스타일(최우선)
    - 저시력자나 색약자와 같이 특별한 조건이 필요한 사용자가
      그들에게 맞게 구성해 놓은 스타일 시트
    - 예를 들면 저시력자가 글자를 명확히 읽기 위해 
      윈도우의 고대비 설정 기능을 이용하면
      검정색 배경과 흰색 글자로 바뀌고 사용자 시트로 저장된다.
      이 스타일 시트는 시스템을 통해서 만들어진 것이기 때문에
      제작자가 제어할 수 없다.

2. 제작자가 만들 스타일 중 !important가 붙은 스타일
    - 이 시트는 사이트를 제작하면서 만든 스타일 시트로서
      다른 스타일보다 최우선으로 적용해야 할 스타일에
      !important를 붙일 수 있다.
    - 다른 어떤 스타일보다 우선

3. 제작자가 만든 일반 스타일
    - 사이트를 제작하면서 만든 스타일

4. 기본적인 브라우저 스타일 시트
    - 브라워저에 기본적으로 지정한 스타일

[적용범위]
- 스타일 적용이 충돌 될 때 적용 범위
- 스타일 적용 범위가 좁을 주록 우선순위가 높다.
(중요도 우선 순위)
1. 인라인 스타일
2. ID 스타일
3. 클래스 스타일
4. 태그 스타일

position - 배치 방법

position : static | relative | absolute | fixed

  • static: 기본값, 문서의 흐름에 맞춰서 배치
  • relative: 이전 요소에 연결해서 배치
  • absolute: 위치를 지정해서 배치
  • fixed: 고정 배치

static

기본 속성값으로 나열한 순서대로 배치, top, left, right, bottom을 사용할 수 없다. float로 left, right를 사용해서 좌.우 배치 가능

relative

나열한 순서대로 배치하며 top, left, right, bottom 속성을 사용할 수 있다. 또는 좌표값을 사용해서 위치를 지정할 수 있다.

absolute

문서의 흐름과는 관계없이 원하는 위치에 배치하며 left, right, top, bottom 속성 값을 사용해서 원하는 위치에 배치할 수 있다.
기준이 되는 위치는 가장 가까운 부모 요소나 조상 요소 중 position 속성이 relative인 요소이다.

fixed

absolute 처럼 문서의 흐름과는 관계없이 위치를 좌표로 지정할 수 있으며 부모 요소가 아닌 브라우저 창이 기준이 된다(클라이언트 영역 기준).

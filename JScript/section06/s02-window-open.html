윈도우 창 열기 - open()

open(url, 창이름, 옵션)

  • url : 창을 오픈하여 연결할 주소
  • 창이름 : 새 창의 이름, 문자열
  • [옵션]
    • width: 창의 너비
    • height: 창의 높이
    • left: 창의 위치(x축)
    • top: 창의 위치(y축)
    • location: 창의 주소 입력 출력 유무(yes or no)
    • scrollbars : 창의 스크롤바 출력 유무(yes or no)
    • status : 창의 상태 바 출력 유무(yes or no)
    • toolbas : 창의 도구 상자 출력 유무(yes or no)

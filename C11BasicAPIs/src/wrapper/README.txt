2. 포장 클래스(Wrapper Class)
: 기본 타입에 대응하는 클래스로서 각 기본 타입의 값을 가지고 있다.
: 추가적으로 유용한 기능을 제공
: 기본 타입을 객체화 해서 사용
: 기본 타입에 대응하는 객체를 요구하거나 제공하는 경우

- Byte: byte
- Character: char
- Short: short
- Integer: int
- Long: long
- Float: float
- Double: double
- Boolean: boolean

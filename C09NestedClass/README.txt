[중첩 클래스와 중첩 인터페이스]

(목적)
1. 외부에 공개할 필요가 없는 클래스를 내부 클래스로 만들어서 내부에서만 사용하도록 제한
2. 외부로 공개할 부분과 내부에서만 사용해야 할 기능으로 분리
3. 내부에서 사용하는 전용 클래스로 한정

(구조)
1. 멤버 클래스: 클래스 안에 클래스 정의
   - 인스턴스 멤버 클래스 : 객체화 되어야 사용 가능
   - 정적 멤버 클래스 : 객체화가 되지 않아도 사용 가능
2. 로컬 클래스
   - 메소드 안에 클래스 정의
   - 메소드가 실행되어야 사용 가능   
   
(제약사항)
1. 중첩 클래스에서 정적 필드 선언할 수 없다.
   - final static 필드는 가능
2. 정적 메소드는 정의할 수 없다.

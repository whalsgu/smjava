[연산자와 연산식]

1. 연산(Operations)
- 프로그램에서 데이터를 처리하여 결과를 산출하는 것
- 계산 결과를 도출 하는 것

2. 연산자(Operator)
- 연산에서 사용되는 표시난 기호
- 산술연산자(+, -, *, /, %)
- 논리연산자
- 비교연산자

3. 피연산자(Operand)
- 연산되는 데이터

4. 연산식(Expressions)
- 연산자와 피연산자를 이용하여 연산의 과정을 기술하는 것

5. 연산의 방향과 우선순위
- 산술연산방향 : 좌측 -> 우측

비교연산자: ==, !=, >, >=, <, <=
 처리결과는 boolean
 - EQ : 같다(==)
 - NE : 같지안다(!=)
 - GT : 크다(>)
 - GE : 크거나 같다(>=) 
 - LT : 작다(<)
 - LE : 작거나 같다(<=)

[확인문제]

[문제1] 연산자와 연산식에 대한 설명 중 틀린 것은?
1. 연산자는 피연산의 수에 따라 단항, 이항, 삼항 연산자로 구분된다.
2. 비교 연산자와 논리 연산자의 산출 타입은 boolean(true/false)이다.
3. 연산식은 하나 이상의 값을 산출할 수 있다.
4. 하나의 값이 올 수 있는 자리라면 연산식도 올 수 있다.

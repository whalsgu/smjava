[문제11] 접근 제한에 대한 설명으로 틀린 것은?
정답: 3번
해설: default 접근 제한은 해당 클래스와 해당 패키지에서 사용을 허가한다.
      class 접근 제한은 public, default만 사용한다.
      클래스 내부에서만 사용을 허가하기 위해서는 private 지정한다.

1. 접근 제한자는 클래스, 필드, 생성자, 메소드의 사용을 제한한다.
2. public 접근 제한은 아무런 제한 없이 해당 요소를 사용할 수 있다.
3. default 접근 제한은 해당 클래스 내부에서만 사용을 허가한다.
4. 외부에서 접근하지 못하도록 하려면 private 접근 제한을 해야 한다.

[Spring Framework]

(특징)
1. 자바 애플리케이션 프레임워크
2. 경량 프레임워크(Light Weight Framework)
3. 경량 컨테이너(Light Weight Container)
4. 자바 빈(클래스 객체) 스프링 생성, 초기화, 실행, 소멸을 관리
5. 제어의 역전(IoC, Inversion of Control)
	- 프레임워크 객체를 관리
	- 애플리케이션 사이의 느슨 결합
6. 의존성 주입(DI, Dependency Injection)
	- 프레임워크가 객체를 생성하고 값을 지정
7. 관점지향(AOP, Aspect Oriented Programming)
	- 핵심 기능과 부가 기능을 분리
8. POJO	
	- Plain Old Java Object
	- 순수 자바 객체
	
	
(구조)
1. Spring Core
	- IoC
2. Spring AOP
	- 관점 지향 기능 제공
3. Spring DAO
	- JDBC
4. Spring Context
	- Bean에 접근 방법 제공
5. Spring ORM
	- 하이버네이트, 마이바티스 같은 영속성 관련 연동 기능 제공
6. Spring Web
	- 웹 애플리케이션 기능 제공
7. Spring MVC
	- MVC 구현 기능 제공


(lib)
1. spring.jar
2. commons-logging.jar

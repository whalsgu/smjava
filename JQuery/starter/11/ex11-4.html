웹이란?

웹은 "World Wide Web"의 약어로서 간단하게 WWW로 표현한다. 웹이란 인터넷에 연결된 컴퓨터를 통해 전 세계 사람들이 정보를 제공하고 공유할 수 있는 사이버 공간을 의미한다.

웹 브라우저

웹 사이트에 구축된 웹 페이지, 즉 HTML 문서를 볼 수 있는 응용 프로그램을 의미한다.인터넷 익스플로러, 크롬, 사파리, 파이어폭스 등의 프로그램이 여기에 속한다.

웹 호스팅

웹 사이트에 구축된 웹 페이지, 즉 HTML 문서를 볼 수 있는 응용 프로그램을 의미한다.인터넷 익스플로러, 크롬, 사파리, 파이어폭스 등의 프로그램이 여기에 속한다.

웹 프로그래밍

웹 사이트의 기능을 구현하기 위하여 HTML/CSS와 웹 프로그래밍 언어(자바스크립트, PHP, JSP 등)를 이용하여 프로그램을 작성하는 것을 의미한다.

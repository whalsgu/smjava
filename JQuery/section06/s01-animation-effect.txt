[제이쿼리에서 애니메이션 효과]

(숨김)
1. hide() 요소를 숨김
2. fadeOut() 요소가 점점 투명해지면서 사라짐
3. slideUp() 요소가 위로 접히면서 숨겨짐

(노출)
1. show() 숨겨진 요소가 노출
2. fadeIn() 숨겨진 요소가 점점 선명해 짐
3. slideDown() 숨겨진 요소가 아래로 펼쳐짐

(노출, 숨김)
1. toggle() 숨김(hide), 노출(show) 효과 적용
2. fadeToggle() fadeIn, fadeOut 효과 적용
3. slideToggle() slideUp, slideDown 효과 적용
4. fadeTo() 지정한 투명도 적용

(효과 메서드 기본형)
$("요소선택).효과메서드(효과소요시간, 가속도, 콜백함수);
효과소요시간: 밀리세컨드(1000분의 1초), "slow", "normal", "fast"
가속도: 
    "linear" : 일정한 속도로 움직임
    "swing"  : 시작과 끝은 느리게, 중간은 빠르게 움직임(기본값)
콜백함수: 생략가능, 효과가 끝난 후 실행할 함수    

(애니메이션)
1. 애니메이션 메서드는 함수가 적용된 순서대로 큐(queue)에 저장
2. 큐(Queue), FIFO(First In First Out), 선입선출
3. 애니메이션 메서드
   $("요소선택).animate({스타일속성}, 적용시간, 가속도, 콜백함수);
   가속도: 
       "linear" : 일정한 속도로 움직임
       "swing"  : 시작과 끝은 느리게, 중간은 빠르게 움직임(기본값)
   콜백함수: 생략가능, 애니메이션이 끝난 후 실행할 함수   


(애니메이션 효과 제어 메서드)
1. stop() 현재 실행 중인 효과를 모두 정지
2. delay() 지정한 시간만큼 지연했다가 진행
3. queue() 큐에 사용자 정의 함수를 추가하거나 큐에 대기중인 함수를 배열에 담아서 반환, 애니메이션 효과는 모두 취소
4. clearQueue() 큐에 처음으로 진행하고 있는 애니메이션만 제외하고 대기 중인 애니메이션은 모두 제거
5. dequeue() 큐(queue)에 의해서 제거 되고 있는 애니메이션의 메서드의 제거를 막음
6. finish() 선택한 요소의 진행 중인 애니메이션을 강제로 완료 시점으로 보낸 후 종료

(애니메이션 사용법)
$("요소선택").stop();   진행중인 애니메이션만 정지, 큐에 대기중인 애니메이션은 계속 실행

$("요소선택").stop(clearQueue, finish); 
finish : true, false(기본값)
  - true : 큐에서 대기중인 애니메이션을 모두 제거, 강제 종료
